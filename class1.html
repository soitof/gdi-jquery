What is a library?

A collection of reusable methods for a particular purpose.

Code that:

  • someone else wrote
  • you can use
  • included with your code

A math library could have


math.sum(array);

Life without jQuery

Lets try some dom manipulation without the use of jquery, but using a different library.

Exercise time

teaching-materials.org/ jquery/exercises/videoplayer_library.html

What do we need to do?

  • Make an array of the links on the page using document.getElementsByTagName(tagName)
  • Iterate through the array. In each iteration of the loop:
    • Find the current href
    • Generate a thumbnail URL
    • Create an IMG element
    • Set the "src" of the IMG element
    • Append the IMG to the link

How can we?

  • Array of the links: document.getElementsByTagName(tagName)
  • Iterate through the array. In each iteration of the loop:
    • Current href: element.getAttribute(name)
    • Thumbnail URL: youtube.generateThumbnailUrl(videoUrl)
    • IMG element: document.createElement(tagName)
    • Set "src": element.setAttribute(name, value)
    • Append the IMG: element.appendChild(element)

The code


var videoLinks = document.getElementsByTagName('a');
for (var i = 0; i < videoLinks.length; i++) {
	var linkUrl = videoLinks[i].getAttribute('href');
	var thumbnailUrl = youtube.generateThumbnailUrl(linkUrl);
	var thumbnailImg = document.createElement('img');
	thumbnailImg.setAttribute('src', thumbnailUrl);
	videoLinks[i].appendChild(thumbnailImg);
}

jQuery

The most popular library.

Write less code for more common tasks.

Abstracts certain functionality for cross browser support.

Open source, big community.

What does it handle

Start with something basic


<p>Welcome to jQuery!</p>

Add a class


$('p').addClass('special');

Results in


<p class="special">Welcome to jQuery!</p>

What is going on in this important bit.


$('p').addClass('special');

Break it into three parts.

1: dolla dolla bill y'all.


$

The global jQuery function.

Tell jQuery to do your bidding.

2: get it all.


('p')

Finds DOM element(s) according to what's in the quotes.

Returns a "jQuery collection."

3: make it special.


addClass('special')

Built-in jQuery method that adds the specified class to the collection.

Adds it to all memebers of the collection.

Find all kinds of things!

All selectors are valid.

<p class="intro">Welcome!</p>

Get at it:

You've got the idea:

Creation

Not only can you edit what is in the dom, you can add to it as well.

Create elemets in the dom with jQuery.

Pass in an HTML string and jQuery will create the element for you.

The result is also returned as a collection.

jQuery:


$('<p>');

HTML:


<p></p>

What you ask:


$('<p>Welcome!</p>');

What you get:


<p>Welcome!</p>

Witchcraft:


$('<p class="intro">Welcome!</p>');

Black magic:


<p class="intro">Welcome!</p>

So what?

Creating things is cool, but doesn't look super useful.

It is, once we start to store these elements and use them later.

jQuery returns collections when it perfoms it's functions.

We can save the return of these functions and use them as needed.

Created our welcome paragraph and save it off


var myParagraph = $('<p class="intro">Welcome!</p>');

Once we got it, flaunt it!

Inject it into other parts of the page.


$('body').append(myParagraph);
					

$('body').prepend(myParagraph);
					

myParagraph.appendTo($('body'));

Loops

Looping is something you constantly do.

jQuery makes it easy to loop over the collections it gives.


var links = $('a');

links.each(function(index, elem) {
   var link = $(elem);
   link.html(link.html() + '!');
});
					

var links = $('a');
var processElem = function(index, elem) {
   var link = $(elem);
   link.html(link.html() + '!');
};

links.each(processElem);
