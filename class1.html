What is a library?

A collection of reusable methods for a particular purpose.

Code that:

  • someone else wrote
  • you can use
  • included with your code

A math library could have


math.sum(array);
math.pow(num, num);
math.factorial(num);

How would I use this?


<script src="http://imagine-it.org/math.js"></script>
<script>
	var answer = math.sum(2, 2);
	alert(answer);
</script>
